⚙️ ¿Cómo cambiar la configuración?

La configuración del bot se gestiona a través de las variables de entorno en Railway. Para actualizarla:

  1. Ve a tu proyecto en railway.app
  2. Selecciona tu servicio
  3. Haz clic en Variables
  4. Edita los valores correspondientes
  5. Railway reiniciará el servidor automáticamente
Variables disponibles:
BARBERIA_NOMBRENombre de la barbería en el bot
BARBERIA_DIRECCIONDirección que ve el cliente
BARBERIA_HORA_APERTURAHora de apertura (ej: 09:00)
BARBERIA_HORA_CIERREHora de cierre (ej: 20:00)
SERVICIO_PREDETERMINADOServicio que se asigna en citas del bot
BARBEROS_MOSAICO_URLURL de imagen del equipo (opcional)
JEFE_BARBERO_TELEFONOTeléfono para notificaciones (ej: 573001234567)
ADMINISTRADORA_TELEFONOTeléfono admin para notificaciones
ADMIN_USERNAMEUsuario para este panel
ADMIN_PASSWORDContraseña para este panel
